    <h1>Introducción</h1>

    <p>El proyecto trata sobre la creación de un sistema para la cadena de supermercados <span class="jumbox">JUMBOX</span>. Este completo sistema se va a conectar y podra interactuar con una base de datos, y ofrece una amplia gama de funciones diseñadas para simplificar y potenciar la gestión del supermercado.</p>

    <p>Entre las características destacadas del sistema encontramos su capacidad para administrar usuarios de manera efectiva, el control de stock, la gestión de pedidos y la capacidad de generar informes detallados.</p>
